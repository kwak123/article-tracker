# kwak-react
Simple Article Tracker to help me trace blogs to read

## Notes
* Node must be >= 9.0.0

## Starting
Run commands
1. `yarn install`
2. `yarn react-dev`
3. `yarn start`

Note, you will need to run yarn start in a new thread
